\section{IIB}
Despite there are various conventions one may adopt in choosing the Einstein frame action, where the SL(2, R) structure is manifest, there is only one string frame action. 
But in the string frame, the SL(2, R) invariance is not manifest.
The 11d to 10d reduction is often performed in the string frame, which elucidates the interpretation of the various fields in the higher dimension.
If the type IIB action were to be a compactification of a higher dimensional theory, perhaps it would be related to the string frame action, but SL(2, R) invariance should be manifest.

the type IIB supergravity action in the string frame is


\begin{equation}
\begin{aligned}
S_{IIB}
&=S_{NSNS} + S_{RR} + S_{CS}\\
&=\frac{1}{(2\pi)^7\alpha^{\prime 4}}
\int d^{10}x \sqrt{-g}
\Bigg[
e^{-2\Phi}\left(R + 4(\partial\Phi)^2-\frac{1}{2}|H_3|^2\right)
-\left(\frac{1}{2}(\partial C)^2+\frac{1}{2}|\tilde F_3|^2+\frac{1}{4}|\tilde{F}_5|^2\right)
\Bigg]\\
&\quad -\frac{1}{2(2\pi)^7\alpha^{\prime 4}}\int C_4\wedge H_3\wedge F_3
\end{aligned}
\label{string frame IIB action}
\end{equation}
with $(2\pi)^7\alpha^{\prime 4}=(2\pi)^7l_s^8=(2\pi)^7\alpha^{\prime 4}$, and
\begin{equation}
F_p = dC_{p-1}, \quad 
H_3 = dB_2, \quad
\tilde F_3 = F_3 - C H_3,\quad
\tilde F_5 = * \tilde F_5 = F_5 - \frac12 C_2\wedge H_3 + \frac12 B_2\wedge F_3.
\end{equation}
It is a quick computation to show that the string frame metric transforms under SL(2, Z) as
\begin{equation}
g_{mn}\to |c\tau + d| g_{mn}.
\end{equation}
It may also be helpful to remember that under SL(2, Z),
\begin{equation}
e^{-\Phi} \to \frac{e^{-\Phi}}{|c\tau + d|^2},\quad
\begin{pmatrix}
a & b\\
c & d
\end{pmatrix}
\in SL(2, Z).
\end{equation}
