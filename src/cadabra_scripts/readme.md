# Objective

There are some interesting results related to complicated derivations and dimensional reductions of higher-derivative effective actions of the type IIB supergravity as well as its uplift to 12d. 

Specifically, the objective is to find 12d interpretations of t10d effective actions.
At four point for the axio-dilaton sector, this has already been done, in 1506.06757 (and also followed up in 2602.19979). Recently, there has been some proposals for 12d uplifts of the 3-form sector of type IIB into 12d 4-form (2512.10746), so a natural question is whether a ansatze for 12d effective action involving 4-form could reproduce 10d 3-form effective actions, that's the objective here.

The first step to do so is to reproduce the axio-dilaton results in 1506.06757 as well as 2602.19979. 
These results of 1506.06757 as well as 2602.19979 are likely derived using CAS but the code for them is not available.
So as a first step I would like to write the Cadabra code to reproduce them.
